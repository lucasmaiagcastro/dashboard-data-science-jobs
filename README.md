# Dashboard de Vagas em Ciência de Dados

Projeto desenvolvido para a disciplina de Visualização de Dados da Unichristus.

## Objetivo

Analisar o mercado de trabalho em Ciência de Dados utilizando visualizações interativas construídas com Dash e Plotly.

## Tecnologias

- Python
- Dash
- Plotly
- Pandas
- GitHub
